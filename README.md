# Lojinha Web — Sistema de Loja Virtual (Projeto Web)

Este repositório contém o código-fonte do projeto **Lojinha Web**, uma aplicação web desenvolvida com o objetivo de simular o funcionamento de uma loja virtual (e-commerce).  
O projeto foi criado para fins acadêmicos e de estudo, com foco no aprendizado prático de desenvolvimento web, organização de código e estruturação de sistemas.

## Status do Projeto

Em desenvolvimento.  

O projeto encontra-se funcional em sua proposta básica, porém segue aberto para melhorias, refatorações e inclusão de novas funcionalidades conforme a evolução do aprendizado e dos requisitos do sistema.

## Contexto e Motivação

A criação da Lojinha Web teve como motivação principal o aprimoramento de conhecimentos em desenvolvimento web, especialmente na construção de interfaces, organização de arquivos, lógica de aplicação e boas práticas de código.

Além do caráter acadêmico, o projeto também foi pensado como parte de portfólio, demonstrando a capacidade de planejar, estruturar e documentar uma aplicação web do zero.

## Descrição Geral do Sistema

A Lojinha Web representa uma loja virtual simples, com foco na apresentação de produtos e na simulação de um ambiente de e-commerce.  
O sistema foi projetado para ser claro, intuitivo e de fácil manutenção, permitindo futuras expansões como:

- Carrinho de compras
- Controle de pedidos
- Área administrativa
- Integração com backend e banco de dados

## Objetivos do Projeto

- Simular o funcionamento básico de uma loja virtual  
- Praticar conceitos fundamentais de desenvolvimento web  
- Trabalhar organização de projeto e separação de responsabilidades  
- Desenvolver interfaces simples, funcionais e responsivas  
- Criar uma base sólida para futuras evoluções do sistema  

## Funcionalidades

### Funcionalidades Implementadas

- Página inicial da loja  
- Estrutura de listagem de produtos  
- Layout organizado e intuitivo  
- Separação de arquivos por responsabilidade (HTML, CSS e JavaScript)  

### Funcionalidades Planejadas

- Sistema de carrinho de compras  
- Página de detalhes do produto  
- Simulação de finalização de compra  
- Validação de formulários  
- Responsividade aprimorada para dispositivos móveis  
- Integração com backend  
- Persistência de dados em banco de dados  

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

### Front-end
- HTML5  
- CSS3  
- JavaScript  

### Ferramentas de Desenvolvimento
- Visual Studio Code  
- Navegador Web (Google Chrome, Firefox, Edge, etc.)  

Outras tecnologias e frameworks poderão ser incorporados conforme a evolução do projeto.

## Estrutura do Projeto

A estrutura de diretórios foi organizada de forma a facilitar a leitura, manutenção e escalabilidade do sistema:

```

/
├─ assets/                Imagens, ícones e recursos visuais
├─ css/                   Arquivos de estilo
│  └─ style.css
├─ js/                    Scripts JavaScript
│  └─ script.js
├─ index.html             Página principal da loja
├─ README.md              Documentação do projeto

````

Essa estrutura pode ser expandida conforme novas funcionalidades forem adicionadas.

## Arquitetura e Organização

O projeto segue uma separação básica de responsabilidades:

- HTML: estrutura das páginas  
- CSS: estilização e layout  
- JavaScript: comportamento e lógica da aplicação  

Essa separação facilita a manutenção do código e a futura integração com frameworks ou backend.

## Como Executar o Projeto Localmente

### Pré-requisitos

- Navegador web atualizado  
- Editor de código (recomendado: Visual Studio Code)

### Passo a Passo

1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
````

2. Acesse a pasta do projeto:

   ```bash
   cd lojinha-web
   ```

3. Abra o arquivo `index.html` diretamente no navegador
   ou utilize a extensão Live Server no Visual Studio Code para execução local.
